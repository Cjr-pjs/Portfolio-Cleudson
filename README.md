## 📦 Instalação

### Pré-requisitos

Antes de começar, certifique-se de ter instalado:

- [Node.js](https://nodejs.org/) `>= 18.x`
- [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/) ou [pnpm](https://pnpm.io/)
- [Git](https://git-scm.com/)

### Passo a passo

**1. Clone o repositório**

```bash
git clone https://github.com/seu-usuario/portfolio-cleudson.git
```

**2. Acesse a pasta do projeto**

```bash
cd portfolio-cleudson
```

**3. Instale as dependências**

```bash
npm install
# ou
yarn install
# ou
pnpm install
```

**4. Inicie o servidor de desenvolvimento**

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
```

**5. Acesse no navegador**

```
http://localhost:3000
```

## 🎮 Uso

### Scripts disponíveis

| Comando | Descrição |
|---------|-----------|
| `npm run dev` | Inicia o servidor de desenvolvimento |
| `npm run build` | Gera a build de produção |
| `npm run start` | Inicia o servidor em modo produção |
| `npm run lint` | Verifica erros de lint no código |

### Personalizando o portfólio

As principais configurações ficam em:

```
portfolio-cleudson/
├── app/                  # Páginas e layouts (App Router)
├── components/           # Componentes reutilizáveis
├── public/               # Imagens e arquivos estáticos
│   └── screenshots/      # Screenshots para o README
├── styles/               # Estilos globais
└── data/ (ou config/)    # Seus dados pessoais, projetos, etc.
```
